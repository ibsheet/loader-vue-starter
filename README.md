# ibsheet-loader-vue-starter

[`@ibsheet/loader`](https://github.com/ibsheet/loader) starter for vue

## Clone Project

```bash
git clone https://github.com/ibsheet/loader-vue-starter ibl-vue-starter
cd ibl-vue-starter
```

## Copy IBSheet Library

copy to: `public/ibsheet`

## Start

### Using npm:

```
npm install
npm run serve
```

### Using yarn:

```
yarn
yarn serve
```
